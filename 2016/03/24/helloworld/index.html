开始我的博客

今天跟大神交流了一下，看到他通过Github创建了个人博客，不花钱又省掉管理的麻烦，于是厚着脸皮
让他教了我怎么搭建，所以就有了这第一篇博客了。

首先献上参考链接，大学同学的博客，大神一个: http://blog.zjy.space

准备工作

首先你要有Github账号，node.js，git bash
注册和安装应用的流程就省略了
提供地址
Github node.js git bash

步骤

第一步 创建仓库

在github上创建新的仓库，在输入项目名称时要注意，Repository name这一栏要一下格式填写

你的用户名.github.io 这样就可以以网站的形式访问你的github仓库，访问名就是这个。
据说之所以能够这样做是github为了给开发者更好地介绍自己的项目特意设置的，注意每个用户
只能有一个这样命名和访问的仓库，所以一般拿来搭建个人介绍博客。

第四步 添加第一篇博客

双引号里边是博文标题，请自己决定
然后就可以在目录下source->_posts找到新建的文件 firstblog.md
hexo的每一篇博文都是以.md的格式保存，所以需要了解一下markdown的语法
简单说明一下开头

还有别的我还没学就先不介绍了，title是标题，date是发布时间，tags是标签，
hexo用于自动检索同标签文章，每个标签以- 标签名注意中间空格，category是分类。
开头过后就开始写自己的博客了，除了md语言，也同时支持html和js的使用，实际上直接解析成网页了。
所以用不习惯的可以直接打html。
